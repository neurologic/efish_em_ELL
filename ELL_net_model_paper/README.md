# MATLAB Project README

## Overview

This repository contains MATLAB scripts and data files developed for analyzing and simulating experimental data and computational models for the paper entitled 'Connectome analysis of a cerebellum-like circuit for sensory prediction'. The code is modular, commented, and designed for easy reproduction and adaptation. To use it please download and uncompress the file 'ELL_net_model_paper.zip'.

## Contents

- The folder 'utilities_ell_net' contains all the necessary functions.
- The folder 'data_ell_net' contains all the necessary data files.
- The main folder contains the functions to run. The file name refers to the relevant figures in the paper it generates. At the beginning of each code there is an explanation of the necessary parameters to manipulate if any.

## Requirements

- The code was generated using Matlab version 2024b.

## Usage

1. **Clone or download** the repository (see efish_em_ELL README: Install).
2. Add the repository (or relevant folders) to your MATLAB path:
3. Run the main scripts.
4. Results will be displayed as figures.

## Data

All `.mat` files in the `EM_data_published/data_ell_net/` directory contain variables used in the scripts. The relevant scripts includes documentation describing how the data is loaded and used.
